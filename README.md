Este projeto consiste em um site educativo desenvolvido com o objetivo de apresentar informações sobre os tipos de rochas: magmáticas, sedimentares e metamórficas. O conteúdo aborda a formação, as características e algumas curiosidades sobre cada tipo, de forma clara e acessível.

O site foi criado como projeto final, com foco em aprendizado e aplicação prática de desenvolvimento web.

Tema do Projeto

O tema abordado é Tipos de Rochas, conteúdo estudado na área de Geografia e Ciências da Terra. A proposta do site é facilitar o entendimento do assunto por meio de uma navegação simples e visual.

Tecnologias Utilizadas

HTML5 – Estruturação do site

CSS3 – Estilização e layout

JavaScript – Interatividade e dinamismo

GitHub Pages – Hospedagem do site
